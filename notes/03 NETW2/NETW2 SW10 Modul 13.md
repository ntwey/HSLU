# Module 13 – WLAN Configuration

## 1. WLAN Konfiguration im Remote-Site Router
- Standard IP ändern
- Admin-Passwort ändern
- DHCP-Bereich anpassen
- SSID, Channel, Security einstellen
- WPA2/WPA3 verwenden
- Optional: Mesh-Netzwerke, QoS, Port-Forwarding

## 2. WLAN im WLC konfigurieren
### WLC Grundlagen
- Lightweight APs via CAPWAP
- Zentrale Verwaltung vieler APs

### WLAN erstellen (Beispielschritte)
1. Neue WLAN-ID + SSID anlegen  
2. Aktivieren  
3. Schnittstelle zuweisen  
4. Sicherheit konfigurieren (WPA2 PSK/Enterprise)  
5. Monitoring & Client-Übersicht

### WPA2 Enterprise
- RADIUS Server
- VLAN-Zuordnung
- DHCP Integration

## 3. Troubleshooting
- Verbindung prüfen
- DHCP/Radius Probleme lösen
- AP Join-Issues (Discovery, Join, Image Download)
