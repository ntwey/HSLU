## **Modul 1: Basic Device Configuration**

> **Ziel:** Geräte mit Sicherheitsbest Practices konfigurieren.

### Themen:

- **Switch Boot Sequence:** POST → Bootloader → CPU Init → Flash Init → IOS laden.
    
- **BOOT-System:** `boot system flash:<pfad>` legt Startdatei fest.
    
- **Switch LEDs:** Anzeigen für System, RPS, Duplex, Speed, PoE.
    
- **Recovery Mode:** Zugriff über Bootloader bei Systemfehler.
    
- **Management Access:** IP + Subnet für SVI (Switch Virtual Interface) konfigurieren.
